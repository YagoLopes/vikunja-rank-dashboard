# Vikunja Rank Dashboard

Dashboard simples em Vanilla JS + Vite para gamificação do Vikunja.

## Instalação

```bash
npm install
npm run dev
```

## Token API

No Vikunja:

Settings -> API Tokens

Copie o token e coloque em:

```js
const API_TOKEN = 'SEU_TOKEN'
```

## Labels de Pontos

Use labels assim:

- 5pts
- 10pts
- 20pts

O dashboard soma automaticamente os pontos.
